Web標準サポート

P_BLOGは、W3C勧告の「XHTML 1.0 Strict」及び「XHTML 1.1」に準拠するXHTMLコードを出力するように開発しています。 また、付属のCSSも同様に「W3C勧告のCSS2」に準拠するコーディングをするようにしています。

準拠している標準コーディング規約

PHP 標準コーディング規約

P_BLOGのPHPコーディングはPHP:PEARの標準コーディング規約に準拠しています。

PEAR :: Manual :: Coding Standards

CSS 標準コーディング規約

P_BLOGのCSSプロパティの順序は、Mozilla.orgの基本CSSの「推奨プロパティオーダー」に準拠しています。
